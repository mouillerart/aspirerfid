Sun SPOT Sensors Producer

This project contains the onSPOT Spotlet sensors producer
The spotlet/midlet reads measurements and listens thresholds on the built-in devices (light detector, temperature, 3D-accelrometer, LEDs, switches) and broadcasts the value in a radio message

Authors: Didier Donsez, Lionel Touseau
